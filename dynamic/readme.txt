Features:

- Dynamic dropdown populates car types
- Choosing car type from dropdown displays price and availability
- "Rent Car" Button: 
        - adds one to car booked
        - adds renters name and car type they booked to array "rentals.renters"
        - adds price of car picked to revenue 
        - alerts user if name not entered
        - updates availability
- Car type is removed from dynamic dropdown after all are booked
